# Ninja Adventure

A kawaii ninja platformer game built with pure HTML5 Canvas and vanilla JavaScript. No dependencies, no build tools -- just open and play.

**Play now:** [ninjamario.vercel.app](https://ninjamario.vercel.app)

## Features

- 3 levels: Bamboo Forest, Castle Rooftop, Cherry Blossom Village
- Cute big-eye ninja character with red headband and scarf
- 3 enemy types: Tanuki, Kappa, Oni
- Double jump (ninja arts!)
- Shuriken throwing
- Sushi roll collectibles
- Cherry blossom particle effects
- Web Audio synthesized sound effects (no external audio files)
- Touch controls for mobile + keyboard support

## Controls

| Input | Action |
|-------|--------|
| Arrow Keys / D-pad | Move |
| Space | Jump (press twice for double jump) |
| X | Throw shuriken |
| Enter | Start game |

## Run Locally

```bash
node server.js
# Open http://localhost:3000
```

## Tech Stack

- HTML5 Canvas
- Vanilla JavaScript (ES6+)
- CSS3
- Web Audio API
- Zero dependencies
